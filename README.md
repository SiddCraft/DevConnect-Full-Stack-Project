# DevConnect — MERN Stack Social Platform

Developed by: ELLABOINA SIDDHARTHA  
Domain: Full Stack Web Development (MERN Stack)

## Project Description
DevConnect is a full stack web application designed to connect users through profiles, posts, and interactions. The system demonstrates modern web development using the MERN stack.

## Technologies Used
- React.js (Frontend)
- Node.js & Express.js (Backend)
- MongoDB (Database)
- REST APIs
- JWT Authentication

## Features
- User Registration & Login
- Profile Management
- Post Creation
- Responsive UI
- Secure Authentication
